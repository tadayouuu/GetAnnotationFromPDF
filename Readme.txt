用途:
設計チェック用のPDFに入力されている注釈を抽出してエクセルに転記する。

インストール手順:
1.「Setup.msi」をダブルクリック
2.セットアップウィザードに従い、インストールを行う
3.ユーザーアカウント制御が表示されたら「はい」をクリック
4.（インストールされたら）「閉じる」をクリック

アンインストール手順:
1.「スタート」 > 「設定」 > 「アプリ」
2.「GetAnnotationFromPDF」を選択して「アンインストール」をクリック
3.ユーザーアカウント制御が表示されたら「はい」をクリック
4.「このコンピューターから…」と表示されたら「OK」をクリック

使用手順:
1.「GetAnnotationFromPDF」をダブルクリックして立ち上げる
2.「…」をクリックして、該当のPDFが格納されているフォルダを指定する
3.「実行」をクリックする
処理が完了すると、自動的に該当のフォルダが開く
